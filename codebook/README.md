## 注意事項
1. 標記 0-index or 1-index
2. 以**容易刻錯**或**coding複雜度**為放入優先
3. **黑魔法**次之
4. 若該模板特定區塊需與**另一部份互相配合**，請**註解**
   e.g. 樹鍊剖分需搭配線段樹
5. 變數取名以**看得懂**為主，除非是黑箱內的變數
